0成本搭建Minecraft服务器

前言

上次视频分享了如何用服务器搭建MC服务器
但考虑到有很多学生党是没钱买服务器的
又或者买的服务器配置不够高
所以今天教大家用自己电脑开Minecraft服务器
本期视频主要是面向小白的
所以会比较长一点 还请大家耐心观看
有任何问题也可以在评论区留言 UP尽量回
哦对了 这期视频是介绍Java的开服方式
如果你是基岩版就可以退出视频了
基岩版的话如果有人需要后面也会考虑出一期
好的 那么我们开始正题

安装Java

首先你需要一台电脑 (认真)
然后要弄清你开服务器的版本
MC 1.17及以上 请安装Java 17
1.16及以下 请选择Java 8

Java 17

https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html
这里按照你的系统进行选择
Arm芯片的Mac选择 macOS Arm 64 DMG Installer
Intel的Mac选择 macOS x64 DMG Installer
Windows系统选择 Windows x64 Installer

Java 8

https://www.oracle.com/java/technologies/javase/javase8-archive-downloads.html
Arm的Mac和Intel的Mac都选择 Mac OS X x64
Windows选择 Windows x64
然后安装……
这个应该不用我教了吧
直接双击安装文件一直下一步就好了

下载服务端

MC有很多服务端 我上一期开服的视频也有讲
需要的同学可以自己去我个人主页看上一期的 00:48
这里以Paper为例子
打开 https://papermc.io/downloads
一般下载最新的
我这里以1.18.1为栗
点More 一直到1.18.1
然后点蓝色的按钮下载
那旧版本咋办呢？
往下拉 找到Legacy按钮
点一下 两个问题都选No
点Submit
就可以下载旧版本了
然后我们还要下载一个原版服务端 https://getbukkit.org/download/vanilla 在这里可以下载 注意要和paper的版本一样
这一步也算是完成了

启动服务端

首先 在你熟悉的目录新建文件夹
然后我们把下载好的服务端文件移动到这里
然后Mac用户打开终端
Windows用户的话
Windows 7/8 打开CMD
Windows 10 打开Powershell
Windows 11 打开Windows Terminal
把文件夹拖到终端内 就进入了这个文件夹

然后会卡在下载原版端文件这里
我们把下载好的原版文件 移动到cache文件夹里 改成cache里那个文件的名字 并把原来的删了
然后第一次一般会启动失败 因为你没同意服务器协议
打开 Eula.txt
将false改为true
Ctrl+S 或Command+S 保存
然后再试输入

然后等待出现 Done!字样就说明开服成功了
我们先输入stop关闭服务器
然后可以看到服务端文件夹里生成了很多文件
server.properties是服务器的配置文件
这里online-mode设置为true就代表仅正版玩家可进入 false则盗版玩家也可加入
view-distance为视距 建议改到4
要自定义种子就用Ctrl+F 或Command+F搜索是否有level-seed一行

然后开服就算是完成一半了
我们还需要进行内网穿透

内网穿透

我这里使用的内网穿透软件是Natapp 你也可以用别的
打开Natapp官网 https://natapp.cn/
免费注册账号并实名
点击购买隧道——免费隧道
隧道协议选择TCP
本地端口填25565 也就是MC的默认端口
名称随意 本地地址保持默认
然后复制authtoken 一会要用
点击客户端下载 自行选择系统
一般是Windows 64位
我这里因为是M1 Mac 所以选 MacOS Arm64
然后找个位置存放natapp程序
拖拽到终端进入

点击多人游戏——添加服务器
